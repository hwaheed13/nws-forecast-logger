Release Notes

What's new and improved on Daily Dew Point.

March 22, 2026

Latest
  • Overnight temperature advisory — Dashboard now warns when overnight temperatures may exceed the next day's forecast, helping you anticipate when the daily high could be set overnight rather than during the day.
  • Daily model retraining — The ML model now retrains every morning with the previous day's data, instead of weekly. Predictions adapt faster to changing weather patterns.
  • Enhanced prediction model — Added new atmospheric features to improve accuracy on days with sharp temperature swings.
  • WIN/MISS tracking — ML bucket predictions are now scored against actual results daily, building a transparent track record.
  • Improved Kalshi integration — Live market data now loads reliably with updated API support.

Bug Fixes

  • Fixed Kalshi market data not displaying due to API field update.
  • ML predictions now correctly reflect settled results when daily high is revised overnight.
  • Resolved data pipeline issues affecting prediction accuracy.

March 2026

Earlier this month
  • ML bucket predictions — Launched machine learning model that predicts the most likely 2-degree Kalshi bucket for today and tomorrow.
  • Top-2 bucket display — Shows the two most likely buckets with confidence percentages, giving you a realistic view of the odds.
  • Bet signal indicators — STRONG BET, LEAN, and SKIP labels based on model confidence.
  • Los Angeles support — Added LA (LAX) as a second city with full forecast tracking and ML predictions.
  • Server-side ensemble learning — Forecast weights now learn from historical accuracy and are shared across all users.

Summer–Fall 2025

Foundation
  • Forecast tracking — Began logging every NWS and AccuWeather forecast update with timestamps.
  • Bias correction — Built weighted ensemble that adjusts forecasts based on historical accuracy.
  • Kalshi market display — Live market leaders and settlement results.
  • Historical accuracy stats — Bucket hit rate and forecast accuracy metrics.
